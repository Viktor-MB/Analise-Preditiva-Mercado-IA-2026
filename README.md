# 📈 EDA-Predicao: O Mercado de IA (2020-2026)

Este repositório contém um projeto completo de Ciência de Dados, indo desde a análise exploratória (EDA) até a implementação de modelos de Machine Learning para prever tendências no mercado de trabalho de Inteligência Artificial.

## 🎯 Objetivo
Analisar como o mercado de dados evoluiu nos últimos 6 anos e construir um modelo preditivo capaz de estimar faixas salariais com base em variáveis como senioridade, localização e modelo de trabalho.

## 🛠️ Tecnologias Utilizadas
* **Python 3.x**
* **Pandas & Numpy:** Manipulação e limpeza de dados.
* **Seaborn & Matplotlib:** Visualização de dados e dashboards.
* **Scikit-Learn:** Modelagem preditiva (Regressão/Classificação).
* **KaggleHub:** Integração direta com datasets do Kaggle.

## 📂 Estrutura do Projeto
1. **Limpeza de Dados:** Tratamento de valores ausentes e normalização de moedas.
2. **EDA:** Insights sobre o crescimento de cargos voltados para IA Generativa.
3. **Feature Engineering:** Transformação de variáveis categóricas para o modelo.
4. **Machine Learning:** Treinamento e avaliação de modelos (XGBoost/Random Forest).

## 🚀 Como Executar
1. Clone o repositório: `git clone https://github.com/seu-usuario/EDA-Predicao.git`
2. Execute o notebook: `jupyter notebook EDA-Predicao.ipynb`
